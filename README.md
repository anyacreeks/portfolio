# Any’A S. Creeks | Portfolio Setup Guide

Welcome to my portfolio repository! This guide explains how to set up and customize the portfolio.

---

## ✅ 1. Upload Files
- Ensure these files are in the repository:
  - `index.html` (main portfolio page)
  - `images/` folder (with project images)
  - `ASC_Resume_2026.pdf` (resume for download button)

---

## ✅ 2. Enable GitHub Pages
1. Go to **Settings** in this repository.
2. Scroll down to **Pages**.
3. Under **Source**, select:
   - Branch: `main`
   - Folder: `/root`
4. Click **Save**.
5. Your portfolio will be live at:

https://anyacreeks.github.io/portfolio

---

## ✅ 3. Customize Your Portfolio
- Open `index.html` and edit:
- **About Me** section: Add your bio and education.
- **Projects** section: Update descriptions and add screenshots in `images/`.
- **Contact** section: Verify email and LinkedIn link.
- Replace placeholder images in `images/` with real project visuals.
